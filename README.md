# Welcome to [COMMON CHORDS](https://techhack-production.up.railway.app)

Music has a unique way of bringing people together. A few seconds of a familiar beat can instantly take us back to a memory: a late-night drive with friends, or a quiet moment with headphones after a long day. As a team, we realized nearly all of our favorite memories involving music had something in common: they were shared with other people. Whether it was introducing someone to a new song, recognizing a track within the first few notes, or debating which artist had the better album, those interactions often became meaningful memories.

Test your reaction time, and song recognition with this fun web game!


## Features
- Import your own youtube playlist, or select music by genre or artist.
- Join a shared lobby using a game code.
- Select a set number of rounds.
- Guess the correct track before the timer runs out.
- Receive instant feedback on your guess if it's correct or incorrect.
- Short intermission between rounds to see the correct answer
